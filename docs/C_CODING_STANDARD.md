# C Coding Standard

## 1. Scope

This document defines the C and header file coding rules used by this repository.

## 2. Line Length

- Lines in `.c` and `.h` files should not exceed 120 characters.
- Long structure initializers, macro definitions, function declarations, and function calls should be wrapped across multiple lines.
- Wrapped lines should keep field names, arguments, and macro continuation backslashes readable.
